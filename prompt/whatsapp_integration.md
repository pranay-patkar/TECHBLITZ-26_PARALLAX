**ClinicBrain** is a premium, startup-grade Clinic Management System built for the **TechBlitz26 Hackathon**. It bridges the gap between healthcare providers and patients through a high-performance, real-time architecture and a focus on extreme accessibility.

---

## 🌟 Key Features

* **⚡ Real-Time Booking**: Intelligent scheduling logic that prevents appointment clashes and automatically hides booked slots.
* **🌍 Universal Accessibility**: A comprehensive site-wide translation system (3 languages) designed to help even non-English speaking users navigate with ease.
* **📱 WhatsApp Integration**: Automated appointment receipts sent directly to the patient's registered phone number upon successful booking.
* **🔐 Role-Based Experience**: Customized dashboards for Doctors and Patients, ensuring clear navigation and data security.
* **📊 Integrated Analytics**: Real-time tracking of patient notes, history, and clinic performance.
* **✨ Modern UI/UX**: A visually stunning landing page featuring an automatic horizontal-scrolling quotes banner for enhanced patient engagement.

---

## 🛠 Tech Stack

* **Frontend**: Vite, Vanilla JavaScript, HTML5, CSS3
* **Backend/Database**: Supabase (PostgreSQL)
* **Deployment**: Vercel
* **Version Control**: GitHub

## 🚀 Local Setup

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/VIHAR2212/Clinicbrain.git](https://github.com/VIHAR2212/Clinicbrain.git)
    ```

2.  **Install Dependencies**
    ```bash
    cd Clinicbrain
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
